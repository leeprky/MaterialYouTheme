## How To Install Dusk?

Its Simple!

For **Powercord** Users:

1. Open CMD & Type:

```
cd powercord/src/Powercord/themes && git clone https://github.com/leeprky/Dusk
```

For **Betterdiscord** Users:

1. CLick > [Download](https://raw.githubusercontent.com/leeprky/Dusk/main/source/support/betterdiscord/dusk.theme.css)
2. Download Raw

For **Goosemod** Users:

1. Search The Store
2. Import
